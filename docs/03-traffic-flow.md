# 3. 流量处理完整链路

## 3.1 TCP 流处理

```
┌──────┐     ┌──────────┐     ┌────────┐     ┌──────────┐     ┌─────────┐
│Client │     │  br0    │     │TC BPF │     │ TProxy  │     │ sing-box│
│  :N   │     │ (Bridge)│     │  Hook │     │ :1080   │     │Outbound │
└──┬───┘     └──────────┘     └────────┘     └──────────┘     └─────────┘
   │               │               │               │               │
   │  SYN ────────►│               │               │               │
   │               │  ingress      │               │               │
   │               │──────────────►│               │               │
   │               │               │               │               │
   │               │               │  识别 dst→国外 │               │
   │               │               │  设置 fwmark=1 │               │
   │               │               │  bpf_redirect  │               │
   │               │               │── ─ ─ ─ ─ ─ ─►│               │
   │               │               │               │               │
   │               │               │          TProxy 接收         │
   │               │               │    获取 SO_ORIGINAL_DST      │
   │               │               │── ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─►│
   │               │               │               │               │
   │               │               │          sing-box 建立出站连接 │
   │               │               │               │               │
   │ SYN-ACK ◄────│◄──────────────│◄──────────────│◄──────────────│
   │               │               │               │               │
   │ ACK ────────►│──────────────►│──────────────►│──────────────►│
   │               │               │               │               │
   │ Data ◄──────►│◄─────────────►│◄─────────────►│◄─────────────►│
   │               │               │               │               │
   │ FIN ────────►│──────────────►│──────────────►│──────────────►│
   │               │               │               │               │
   │ FIN-ACK ◄────│◄──────────────│◄──────────────│◄──────────────│
   │               │               │               │               │
```

**关键词**：`SO_ORIGINAL_DST` 用于获取原始目标地址；
TProxy 模式通过 `IP_TRANSPARENT(SO_TPROXY)` 选项实现透明绑定

## 3.2 UDP 流处理

```
┌──────┐     ┌──────────┐     ┌────────┐     ┌──────────┐     ┌─────────┐
│Client │     │  br0    │     │TC BPF │     │ TProxy  │     │ sing-box│
│  :N   │     │ (Bridge)│     │  Hook │     │ :1080   │     │Outbound │
└──┬───┘     └──────────┘     └────────┘     └──────────┘     └─────────┘
   │               │               │               │               │
   │  UDP Pkt ────►│               │               │               │
   │               │  ingress      │               │               │
   │               │──────────────►│               │               │
   │               │               │               │               │
   │               │               │  识别 dst→国外 │               │
   │               │               │  非 DNS 流量  │               │
   │               │               │  设 fwmark=1  │               │
   │               │               │── ─ ─ ─ ─ ─ ─►│               │
   │               │               │               │               │
   │               │               │  UDP TProxy 接收             │
   │               │               │  IP_RECVORIGDSTADDR          │
   │               │               │  获取原始目标地址             │
   │               │               │── ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─►│
   │               │               │               │               │
   │               │               │          sing-box 处理 UDP    │
   │               │               │               │               │
   │ UDP Reply ◄──│◄──────────────│◄──────────────│◄──────────────│
   │               │               │               │               │
```

**UDP 特殊处理**：

| 问题 | 解决方案 |
|------|---------|
| UDP 无连接状态 | 通过 conntrack 维护 UDP 会话 |
| UDP DNS 独立分流 | DNS 流量被劫持到 DNS Subsystem，不经过 TProxy |
| UDP 超时 | 配置 conntrack UDP timeout |
| UDP NAT 行为 | 使用 sing-box 的 UDP over TCP 模式可选 |

### DNS 分流决策树

```
收到 DNS 查询 qname=example.com
│
├─ 检查 DNS 缓存
│  ├─ 命中 → 返回缓存（TTL 内）
│  └─ 未命中 → 继续
│
├─ GeoSite 检查域名归属
│  ├─ CN → 使用国内 DNS（UDP: 114.114.114.114）
│  ├─ 非 CN → 使用国外 DNS（DoH: 1.1.1.1 或 DoT: 8.8.8.8）
│  └─ 未匹配 → 根据默认策略
│
├─ 自定义域名规则覆盖
│  ├─ 强制指定 DNS → 使用指定 DNS
│  └─ 强制代理解析 → 使用国外 DNS
│
├─ 执行 DNS 查询
│
├─ 结果后处理
│  ├─ 过滤污染 IP（例如国外域名返回国内 IP 时丢弃）
│  ├─ 缓存结果
│  └─ 返回给客户端
│
└─ 可选：写入 conntrack 供后续流量匹配使用
```
