## 2.2 五大子系统

### 2.2.1 Control Plane（控制面）

**职责**：系统配置管理、规则引擎、节点管理、健康检查、API 服务

| 模块 | 职责 | 关键接口 |
|------|------|---------|
| Config Manager | 读取/解析/热重载配置 | `config.Load(path)`, `config.Reload()` |
| Rule Engine | 管理分流规则（GeoIP/GeoSite/自定义）| `rule.Match(dstIP, dstDomain) → Action` |
| Node Manager | 管理上游代理节点列表 | `node.List()`, `node.Add()`, `node.Remove()` |
| Health Checker | 周期性检测节点可用性 | `health.Ping(proxyURL) → bool` |
| API Server | HTTP API 供外部管理 | `POST /rule`, `GET /status`, `PUT /config` |

**生命周期**：

```
┌─────────────────────────────────────────────────┐
│  Bootstrap                                        │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐      │
│  │ Config   │→ │ Init     │→ │ Start    │      │
│  │ Load     │  │ Subsys.  │  │ Services │      │
│  └──────────┘  └──────────┘  └────┬─────┘      │
│                                    │            │
│  Runtime                           ▼            │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐      │
│  │ Hot      │  │ Rule     │  │ Health   │      │
│  │ Reload   │  │ Matching │  │ Check    │      │
│  └──────────┘  └──────────┘  └──────────┘      │
│                                    │            │
│  Shutdown                          ▼            │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐      │
│  │ Stop     │→ │ Cleanup  │→ │ Exit     │      │
│  │ BPF      │  │ TProxy   │  │          │      │
│  └──────────┘  └──────────┘  └──────────┘      │
└─────────────────────────────────────────────────┘
```

### 2.2.4 Proxy Engine（代理引擎 — sing-box）

**职责**：作为上游代理 outbound engine，处理转发到 TProxy 的流量

| 组件 | 说明 |
|------|------|
| Outbound Group | 多节点选择器（节点池） |
| Route Router | 基于规则的出站选择 |
| Protocol Adapter | SS / VMess / VLESS / Trojan |

**集成方式**：

```
Marmot Control Plane
       │

       ▼
┌──────────────────┐

```

### 2.2.5 Rule Engine（规则引擎）

**规则匹配优先级**（从高到低）：

1. 自定义 IP 规则（`rules.ip[]`）
2. 自定义域名规则（`rules.domain[]`）
3. GeoSite 规则（基于域名后缀）
4. GeoIP 规则（基于 IP 段）
5. 默认规则（国内 → 直连，国外 → 代理）

**规则数据结构**：

```
Rule
├── Type: "ip" | "domain" | "geoip" | "geosite"
├── Action: "direct" | "proxy" | "block"
├── Target: "CN" | "US" | "custom_list" | ...
├── Priority: uint32 (数值越小优先级越高)
└── Metadata: map[string]string
```
